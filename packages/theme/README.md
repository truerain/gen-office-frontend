### 토큰맵(MAPPING.md)
용도는 “토큰 → 실제 사용 위치”를 빠르게 추적하는 문서입니다.
쉽게 말하면 “이 토큰이 어디서 쓰이는지”를 한 번에 보는 레퍼런스입니다.

#### 핵심 용도
- 영향 범위 파악: 토큰 변경 시 어떤 컴포넌트가 영향을 받는지 즉시 확인
- 리팩터링 가이드: base → semantic 치환 같은 작업에서 “어디를 바꿔야 하는지” 기준점 제공
- 디자인-개발 커뮤니케이션: 토큰 이름과 실제 UI 연결을 명확히 보여줌
- 새 토큰 추가 검증: 새 토큰이 어디에 쓰였는지 확인
   즉, 토큰 관리와 유지보수 효율을 올리는 인덱스 문서라고 보면 됩니다.
